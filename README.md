# FinGuard AI

## An Agentic Financial Discipline Assistant Using NLP, Cognitive Science, Behavioral Economics, and Large Language Models

### Overview

FinGuard AI is an intelligent financial discipline assistant designed to analyze financial thought patterns using Natural Language Processing (NLP), behavioral finance principles, cognitive science concepts, and Large Language Models (LLMs). The system aims to identify financial intent, spending behavior, emotional influences, and cognitive biases that affect financial decision-making.

The application provides personalized financial guidance by combining behavioral analysis with LLM-based reasoning to promote informed and disciplined financial choices.

---

## Objectives

The primary objectives of this project are:

* To analyze financial thoughts using Natural Language Processing techniques.
* To identify behavioral and cognitive patterns influencing financial decisions.
* To detect financial intent such as spending, saving, budgeting, and financial distress.
* To incorporate concepts from behavioral economics and cognitive science into financial analysis.
* To generate context-aware financial recommendations using Large Language Models.

## Methodology

### 1. User Input Layer

The user provides a financial thought or spending-related concern through a text-based interface.

### 2. Natural Language Processing Layer

The system performs textual analysis to classify financial intent into categories such as:

* Spending Behaviour
* Saving Intent
* Budget Planning
* Financial Distress

Keyword-based NLP mechanisms are used to detect financial context and behavioural signals.

### 3. Behavioral Intelligence Layer

The system integrates concepts from behavioral economics and cognitive science to identify financial decision-making biases, including:

* Fear of Missing Out (FOMO)
* Loss Aversion
* Reward-Seeking Behaviour
* Emotional Spending
* Delayed Gratification

### 4. Financial Personality Classification

Based on behavioral patterns, users are categorized into financial personality profiles such as:

* Impulsive Spender
* Emotional Spender
* Smart Saver
* Budget Planner
* Financially Distressed Individual

### 5. AI Recommendation Layer

The system utilizes the Groq API with the Llama 3.3 70B Versatile model to generate context-aware financial recommendations and behavioural guidance.

## Installation and Execution

### Step 1: Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/FinGuard-AI.git
```

### Step 2: Install Required Dependencies

```bash
pip install -r requirements.txt
```

### Step 3: Configure API Key

Insert your Groq API key in the following section of the source code:

```python
api_key="YOUR_GROQ_API_KEY"
```

### Step 4: Execute the Application

```bash
streamlit run app.py
```
